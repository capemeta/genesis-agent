# Genesis Agent

[English](./README.md) | 简体中文

Genesis Agent 是一个基于 Go 的自主 Agent 框架，灵感来自 Claude Code，但目标不是现阶段直接做成成熟的 AI 编程工具，而是面向更通用、更强大的企业级 AI 自动化场景。

项目核心是一个通用 Agent Loop，可通过提示词、工具、MCP、Skills、权限和运行时配置，构建适用于企业任务执行、知识工作、文件处理、命令行工具调用、数据转换、流程自动化和代码辅助任务完成的智能体系统。

这里的“编程能力”主要是作为完成企业级任务的一种手段，例如处理文件、调用命令行工具、转换数据、自动化操作，或在必要时通过代码实现具体任务目标，而不是以构建完整 AI 编程 IDE 或成熟 AI Coding Agent 为当前阶段目标。

Genesis Agent 同时支持企业模式和个人模式：企业模式是基于 React、Ant Design Pro 和 Go 后端 API 构建的企业级系统，提供多租户、RBAC 权限、审计、工具治理、Agent 配置和技能管理；个人模式提供类似 Claude Code 的 CLI，以及桌面端 Agent 工作台。

未来，Genesis Agent 将支持多 Agent 的各种协作范式，包括规划、委派、反思、交接、并行执行、人在回路、长期运行 Agent 等能力；同时支持接入社交软件和企业即时通讯平台，让 Agent 可以通过用户熟悉的沟通入口与人员、团队、工具和业务系统协同工作。

在这些能力的基础上，Genesis Agent 可以进一步支撑企业级 OpenClaw 类产品的构建，用于复杂任务执行、团队协作、业务自动化和智能运营工作流。
