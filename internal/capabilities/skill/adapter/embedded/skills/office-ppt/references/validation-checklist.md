# PPT 验证清单（对齐 Anthropic Visual QA）

**默认有问题。** 第一次渲染几乎从不正确；零问题通常说明看得不够仔细。

## 内容

- 幻灯片数量、顺序和标题符合要求。
- 没有残留占位符、lorem ipsum、示例图或模板说明文字（搜 `xxxx` / `lorem` / `ipsum` / `this.*(page|slide).*layout`）。
- 图表和数字来源清晰，图例和单位完整。

## 视觉（必须看预览图）

用 `render_pptx_preview.py` 逐页图或 `thumbnail.py` 网格，逐项检查：

- 元素重叠（文字穿过形状、线穿过字、多段文字叠在同一 y）
- 文字溢出或在边缘/框边界被裁切
- 表格右侧被裁切（`colW` 之和 + `x` 超过幻灯片宽度）
- 装饰线按单行标题定位，但标题已换行成两行
- 脚注/来源与上方内容碰撞
- 间距过近（小于 0.3"）或卡片几乎相贴
- 间距不均（一处大片空白、一处挤满）
- 距幻灯片边缘不足 0.5"
- 同级栏/卡片未对齐
- 低对比文字或图标
- 文本框过窄导致过度换行、行内挤叠
- 模板修改后母版/字体/品牌规范被破坏

## 闭环

1. 生成 → 转图 → 检查  
2. 列出问题（若没有，再看一遍）  
3. 修复 → 再验证受影响页  
4. **未完成至少一轮 fix-and-verify 不得宣称成功**

截图型幻灯片需要读图中文字时，使用 `office-ocr`。
